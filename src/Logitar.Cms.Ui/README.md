# Logitar CMS UI

This is the React application that serves as a UI for the Logitar CMS. In development, it runs using Vite.js with Node 18.16.0.

## Logitar UI

As of now, Logitar UI is not released yet but is still required in the project. To use it, you need to clone the repository and run `yarn link` in its root folder. Then, in this project, run `yarn link @logitar/logitar-ui` to link the dependency.

## Getting Started

Make sure you have the backend running beforehand.

1. Install Node 18.16.0 and yarn.

2. Install dependencies: `yarn`.

3. Run the development server: `yarn dev`.

4. Open [http://localhost:5173](http://localhost:5173) with your browser.
